クールなハッカーさんたちのためのモダンなブログ環境を作ってみた。
手順は [ここ](http://octopress.org/docs/)の内容を順に追っていただけ。

手元でMarkdownで書いて、

```
$ rake generate
$ rake deploy
```

で投稿できるなんてお手軽でいーですね。

```
$rake preview
```

でWEBrickが立ち上がってプレビューもできます。
